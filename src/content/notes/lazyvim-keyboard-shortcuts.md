# LazyVim File & Directory Management

Essential keymaps for working with files and directories in LazyVim.

---

## File Explorer

| Keymap      | Action                       |
| ----------- | ---------------------------- |
| `<leader>e` | Toggle file explorer         |
| `<leader>o` | Open file explorer (focused) |
| `H`         | Toggle hidden files          |
| `R`         | Refresh file explorer        |
| `a`         | Add file/folder              |
| `d`         | Delete file/folder           |
| `r`         | Rename file/folder           |
| `c`         | Copy file/folder             |
| `m`         | Move file/folder             |

## Quick File Open

| Keymap             | Action                       |
| ------------------ | ---------------------------- |
| `<leader><leader>` | Open file finder (Telescope) |
| `<leader>f`        | Find files in project        |
| `<leader>fg`       | Grep in files                |
| `<leader>fr`       | Recent files                 |

## Buffers

| Keymap       | Action                    |
| ------------ | ------------------------- |
| `<leader>bc` | Close current buffer      |
| `<leader>bo` | Close other buffers       |
| `]b`         | Next buffer               |
| `[b`         | Previous buffer           |
| `<Tab>`      | Switch to next buffer     |
| `<S-Tab>`    | Switch to previous buffer |

## File Operations

| Keymap | Action                   |
| ------ | ------------------------ |
| `:w`   | Save file                |
| `:q`   | Close file               |
| `:qa`  | Close all files          |
| `:wq`  | Save and close           |
| `:qa!` | Close all without saving |
